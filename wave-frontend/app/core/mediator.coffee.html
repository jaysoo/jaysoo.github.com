Signal

A signal can pub/sub on multiple channels.

Example:

signal = new Signal()
signal.subscribe('test', () -> alert('test!'))
signal.publish('test')

    publish: mvc.Events.trigger

include

Provide mvc classes with mixin syntatic sugar

Usage:

Navigation = 
  close: ->
    # Close navigation
  open: ->
    # Open navigation
  toggle: ->
    # Toggle navigation

class Menu extends mvc.Model
  @include Navigation

menu = new Menu()
menu.toggle()

  mediator.include = (mixins...) ->
    throw 'include(mixins...) requires at least one mixin' unless mixins and mixins.length > 0

    for mixin in mixins
      for own key, value of mixin
        @::[key] = value

      included = mixin.included
      included.apply(this) if included
    @

template

Allows a template path to be supplied to a View. Takes in a string argument which is a Handlebars template. Can be used with the text RequireJS plugin.

The @toHTML function will take in JSON and return the rendered HTML.

Usage:

class MyView extends mvc.View
  @template require('text!myapp/templates/myview.html')

  render: =>
    # Render the myview.html template using model's JSON
    @$el.html(@toHTML(@model.toJSON()))
    return @

  mvc.View.template = (text) ->
    @::toHTML = mediator.template(text)
    @

  exports = mediator

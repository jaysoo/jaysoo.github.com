Generating Documentation

Use docco-husky to generate HTML documents from CoffeeScript sources.

Install docco-husky using npm:

$ sudo npm install -g docco-husky

Run the Django collectstatic command:

$ ./manage.py collectstatic --noinput

Generate documentation from the sitestatic/js folder:

$ docco-husky -name "Wave Accounting" `find app -name "*.coffee"`
